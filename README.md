# AI-Based Spectral Fruit Classification

This repository contains deep learning and machine learning approaches for differentiating naturally and artificially ripened fruits using hyperspectral imaging.

## Features
- CNN-based classification
- Handcrafted spectral feature extraction
- SVM classification
- Spectral image analysis
- Transfer learning workflows

## Technologies
- Python
- MATLAB
- Deep Learning
- Computer Vision
- Spectral Imaging

## Results
- Achieved 98.33% classification accuracy
- IEEE APSCON 2026 accepted paper

## Note
Dataset and preprocessing scripts are not publicly included.
